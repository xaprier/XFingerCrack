# Wordlist

Word lists are sourced from [Monkeytype](https://github.com/monkeytypegame/monkeytype) project.

Each JSON file contains:
- `name`: Language identifier
- `bcp47`: Language code (e.g., en-US, tr-TR)
- `words`: Array of words in that language

Source: https://github.com/monkeytypegame/monkeytype/tree/master/frontend/static/languages
